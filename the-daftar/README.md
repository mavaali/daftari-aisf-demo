# The Daftar — Vault README

> **FICTIONAL · DEMO CORPUS — Bureau for Atypical Inquiry**
> All agencies, operations, assets, officers, and persons depicted in this vault are entirely fictional.
> Any resemblance to real intelligence services, persons, or events is coincidental.

The Daftar is the internal knowledge vault of the *Bureau for Atypical Inquiry* — a fictional civilian-cover intelligence service operating from a pre-WWI commercial registry. Its officers call their files *daftars* (ledgers) and their headquarters *the ledger-house*.

This vault is a synthetic demo corpus for the **daftari** knowledge-management tool (Microsoft Agents League @ AI Skills Fest, 2026). It contains 100 canonical files across 8 collections: dossiers, field-reports, ops-active, ops-closed, comp-intel, tradecraft, briefings, and moonshots.

## Getting started

```bash
daftari --vault . --user demo --role director
```

See `.daftari/config.yaml` for role definitions and identity bindings.
